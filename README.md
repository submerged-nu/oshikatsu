## サービス概要
webアプリ「オシカツ！」は推しの好きなところを語れて共有できるアプリです。<br>
キャラクターに焦点を当てており、キャラクターランキングや、おすすめ機能を実装しています。

## 開発背景
好きなキャラクターの絵や動画を作ることが趣味でyoutubeやpixivなどを利用しています。<br>
そこでキャラクターに焦点を当てたアプリがないことに気づき作成しました。
